# Quick Start Guide
Follow these simple steps to get the Rock-Paper-Scissors game running.

## Step 1: Install Python

Make sure you have Python 3.7+ installed:

python --version

## Step 2: Install Dependencies

pip install flask

## Step 3: Run the Server

python main.py

You should see:

Running on http://127.0.0.1:5000

## Step 4: Open Your Browser

Navigate to:

http://localhost:5000 

## Step 5: Play the Game!

Click "Play Round" to play

Watch PalomarA and PalomarB compete After 3 rounds, a winner is announced

Click "New Game" to start over

## Test the API

python test_api.py
